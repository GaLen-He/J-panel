# 更新日志

## v0.8.3 - 2026-07-04

### 新增

- 新增 3DCoat 互导功能，可将 Blender 中选中的网格物体发送到 3DCoat，并从 3DCoat 取回更新后的模型。
- 新增 3DCoat AppLink 交换目录支持：可自动探测默认 Exchange 目录，也可在插件偏好设置中手动指定。
- 新增 3DCoat 互导参数：发送缩放、取回缩放、发送到世界原点、取回时清理无纹理空材质。
- 新增 J Panel 功能菜单，可一键呼出布尔、线框、镜像、3DCoat 互导等常用操作，便于配合 PieMenuEditor 或自定义调用。
- 新增完整弹出面板入口 `qops.show_panel`，可通过快捷键或 PieMenuEditor 呼出完整 J Panel 面板。

### 改进

- 在 3D 视图侧栏 `N > J` 中加入独立的 `3DCoat 互导` 面板，常用互导设置和发送/取回按钮可直接操作。
- 在插件偏好设置中加入 3DCoat 配置区，并保留快捷键列表，方便统一管理快捷键。
- 为 3DCoat 发送、取回和 J Panel 弹出面板注册快捷键条目，默认不占用按键，用户可自行绑定。
- 发送到 3DCoat 前会补齐基础 UV，并按 AppLink 机制写入握手文件，降低 3DCoat 端识别失败的概率。

### 修正

- 调整 3DCoat 往返缩放处理，发送和取回缩放可独立设置，避免模型在 3DCoat 与 Blender 之间来回导入导出后尺寸漂移。
- 取回模型后会清理触发文件，避免重复导入同一个 3DCoat 回传模型。

## v0.5.0 - 2026-07-02

- 初始发布 J Panel。
- 集成布尔运算体选择与显隐控制。
- 集成交互式镜像修改器操作。
- 集成线框显示物体显隐控制。
- 支持在插件偏好设置中管理默认快捷键。
